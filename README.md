# Multisearch
Vanilla Web implementation of a single site to search from multiple providers

Supported providers (first_word):
- Google (google) (Default)
- YouTube (youtube)
- Google Docs (docs)
- Google drive
- Google slides
- Google sheets
- Spotify
- github
- wattpad
- goodreads
- animixplay
- stackoverflow
